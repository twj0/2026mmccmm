========================================================================
MCM 2026 Problem C (DWTS) - Core Source Package
========================================================================

本目录是 2026 MCM C题建模项目的核心代码库，遵循模块化设计原则，旨在实现
从原始数据处理、反向推断建模到策略仿真评估的完整流水线。

------------------------------------------------------------------------
目录结构说明 (Directory Structure)
------------------------------------------------------------------------

.
├── core/           # 核心配置与全局工具
│   └── paths.py    # 项目路径管理，确保在不同环境下文件读取的一致性
│
├── data/           # 数据工程模块 (ETL & Audit)
│   ├── audit.py    # 数据质量审计工具，用于检测缺失值、异常值及赛季覆盖度
│   └── io.py       # 统一的数据读取与持久化逻辑 (支持 CSV/Parquet)
│
├── pipelines/      # 业务流水线 (核心任务实现)
│   # 遵循 mcm2026c_q{N}_{verb}_{object}.py 命名规范
│   # 存放 Q1-Q4 各阶段的具体推断、模拟与分析脚本
│
└── __init__.py     # 将目录标识为 Python 包

------------------------------------------------------------------------
核心设计理念 (Core Philosophy)
------------------------------------------------------------------------

1. 唯一真源 (SSOT):
   通过 core/paths.py 统一管理输入(mcm2026c/)、中间态(data/processed/)
   及输出(outputs/)，禁止在代码中硬编码绝对路径。

2. 模块化解耦:
   - data/ 仅负责数据的形态转换与质量检查。
   - pipelines/ 负责将上述模块串联，完成题目要求的具体任务。
