| 英文专有名词 | 中文对应翻译 | 详细解释 |
|--------------|--------------|-----------|
| Wordle | 沃德乐（音译）/ 单词猜谜游戏 | 《纽约时报》每日推出的热门 5 字母英文猜词益智游戏，玩家需在 6 次及以内尝试猜出目标单词，每次猜测后会收到字母方块颜色反馈，是本题所有数据分析与模型构建的核心研究对象 |
| New York Times | 《纽约时报》 | 总部位于美国纽约市的综合性日报，提供印刷版与网络版服务，是 Wordle 游戏的运营方，也是本题委托分析的需求方，解题结果需向其益智游戏编辑汇报 |
| Twitter | 推特 | 社交网络平台，用户可发布不超过 280 字符的短帖（初始为 140 字符），本题中所有 Wordle 游戏成绩数据均来自该平台用户分享，是数据的核心来源渠道 |
| Hard Mode | 困难模式 | Wordle 游戏的难度模式之一，规则为：一旦玩家找出目标单词中的正确字母（方块显示黄色或绿色），后续所有猜测必须包含这些字母，该模式的玩家占比是本题第二问的研究核心 |
| Regular Mode | 普通模式 | Wordle 游戏的基础模式，无“强制保留已找出正确字母”的规则，与困难模式相对应，是理解游戏玩法差异及玩家选择偏好的关键背景 |
| Contest number | 竞赛编号 | Wordle 谜题的唯一序号标识，自 2022 年 1 月 7 日的第 202 期开始递增，用于关联日期、目标单词与游戏结果数据，是数据整合与溯源的重要字段 |
| Word of the day | 当日目标单词 | 对应特定日期和竞赛编号的 5 字母英文单词，是玩家需猜测的核心目标，其属性（如字母重复度、常见度等）是本题第二、四问的关键分析维度 |
| Number of reported results | 报告成绩总数 | 当日在推特上记录的 Wordle 游戏成绩总条数，该数据的日度波动是本题第一问需解释的核心现象，也是预测 2023 年 3 月 1 日数据的目标变量 |
| Number in hard mode | 困难模式玩家数 | 当日在推特上记录的、选择困难模式进行游戏的玩家成绩数，与“报告成绩总数”结合可计算困难模式占比，是分析单词属性影响的核心指标（第二问） |
| Green tile | 绿色方块 | Wordle 游戏的反馈标识之一，代表该位置的字母在目标单词中存在且位置完全正确，是玩家获取猜测反馈、调整后续策略的关键依据 |
| Yellow tile | 黄色方块 | Wordle 游戏的反馈标识之一，代表该位置的字母在目标单词中存在但位置不正确，与绿色方块共同构成“有效字母反馈”，是困难模式规则的核心触发条件 |
| Gray tile | 灰色方块 | Wordle 游戏的反馈标识之一，代表该字母在目标单词中完全不存在，可帮助玩家排除无效字母、缩短猜测范围 |
| 1 try – 6 tries | 1–6 次尝试 | 玩家在 1–6 次内猜出目标单词的情况，对应数据文件中的玩家占比字段，是第三问的预测目标 |
| 7 or more tries (X) | 7 次及以上 / 未猜出 | 指玩家未能在 6 次及以内猜出目标单词的情况，用 “X” 表示，是难度评估的重要指标 |
| Distribution of Reported Results |成绩分布 | 指 (1,2,3,4,5,6,X) 各结果的概率分布 |
| Solve the Wordle Puzzle | 解出谜题 | 在 6 次或以内正确猜出目标单词 |
| Valid 5-letter Word |有效单词  | 被 Wordle 系统认可的合法 5 字母英文单词 |
| Problem C Data Wordle.xlsx | Wordle 数据文件 | 官方提供的唯一数据源，包含 2022 年全年 Wordle 统计数据，是所有建模分析的基础 |
| COMAP, Inc. | 美国数学及其应用联合会 | MCM 竞赛的主办方，拥有本题版权与数据授权 |
| Prediction interval | 预测区间 | 统计学概念，表示模型预测值可能落入的范围，是第一问的核心输出形式 |
| Word difficulty classification | 单词难度分类 | 基于单词特征将单词划分为简单 / 中等 / 困难，是第四问的核心目标 |
| EERIE | EERIE（指定单词） | 含 3 个重复字母 E 的特殊示例词，用于预测成绩分布与难度分类 |
| Rounding | 四舍五入 | 数据处理中的数值修约操作，可能导致比例和不为 100% |
| Summary Sheet | 摘要页 | 提交论文必须包含的 1 页总结内容，计入 25 页限制 |
| Table of Contents | 目录 | 提交论文的结构导航页，计入 25 页限制 |
| Reference List | 参考文献 | 列出所有引用资料来源，计入 25 页限制 |
