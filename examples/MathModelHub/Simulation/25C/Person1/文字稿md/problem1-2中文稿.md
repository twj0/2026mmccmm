# 问题一、二中文稿

## 论文标题
**数据驱动的奥运奖牌预测与教练效应量化：面向2028洛杉矶奥运会的多模型框架**

---

## 第一章 引言

### 1.1 问题背景

奥运奖牌榜是衡量国家体育实力与资源配置效果的关键指标。2024年巴黎夏季奥运会中：
- **美国**以126枚奖牌（40金）领跑
- **中国**金牌40枚并列第一
- **法国**（东道主）金牌16枚（第5）、总奖牌64枚（第4）
- **英国**金牌14枚、总奖牌65枚列第3

这些结果凸显了经济、人口、传统、项目结构及主场优势的综合影响。

尽管强国格局稳定，巴黎奥运首次夺牌国（如阿尔巴尼亚、佛得角）及超65个从未获奖的国家/地区反映出全球体育成就的不均衡。因此，对2028洛杉矶奥运会的准确预测对各国奥委会的战略规划、突破识别及项目/教练投资至关重要。

### 1.2 问题重述

**任务一**：建立各国2028洛杉矶奥运金牌数与总奖牌数的预测模型，包含不确定性估计与模型性能验证：
1. 预测2028奖牌榜及变化趋势，识别大幅提升或下降概率最高的国家
2. 估算首次获得奥运奖牌的国家数量及概率
3. 分析项目数量/类型与奖牌产出的关系，识别各国关键项目及其原因

**任务二**：分析教练效应（如郎平、卡罗伊）：
1. 提供"伟大教练效应"的证据并估算其对奖牌的贡献量
2. 选择三个国家，推荐优先引进优秀教练的项目，并给出预期奖牌增量估算

### 1.3 建模流程图

建模流程如图所示（`figures/Task1.svg`），包含以下阶段：
- 数据加载 → 数据清洗 → 数据整合 → 特征工程
- 探索性数据分析（目标变量分布、相关性分析）
- 东道主效应验证（描述性统计、t检验、效应量）
- 回归模型构建（线性回归、Ridge、Lasso、随机森林、XGBoost）
- 模型评估与比较（RMSE/MAE/R²）
- 最终预测输出

## 第二章 建模准备

### 2.1 模型假设

1. 历史表现是预测未来表现的有效依据，因为体育基础设施、训练体系和国家优先级随时间持续
2. 东道主效应在不同奥运会之间保持一致，允许从历史数据进行泛化
3. 教练流动效应可通过特定项目中国家奖牌分布的变化观察到
4. 2028年项目总数将与2024年巴黎奥运会大致相似（约329个项目）
5. 政治抵制或全球疫情等外部因素不会显著影响2028年奥运会

### 2.2 符号说明

| 符号 | 描述 |
|------|------|
| $M_{i,t}$ | 国家$i$在$t$年奥运会的总奖牌数 |
| $G_{i,t}$ | 国家$i$在$t$年奥运会的金牌数 |
| $\bar{M}_{k}(i,t)$ | 过去$k$届的奖牌滚动平均 |
| $\text{is\_host}_{i,t}$ | 二值指示变量（1表示东道主，0表示非东道主） |
| $\hat{y}$ | 预测奖牌数 |
| $\beta_j$ | 特征$j$的回归系数 |
| $\alpha$ | 正则化强度参数 |
| $\lambda$ | 泊松分布的平均率参数 |
| $\hat{\tau}_{\text{DID}}$ | 双重差分估计量 |

### 2.3 数据预处理与特征工程

#### 2.3.1 数据来源

本研究使用的数据源自官方提供的五个核心文件，涵盖了从1896年雅典奥运会至2024年巴黎奥运会的历史记录，共30届夏季奥运会、164个国家/地区。

| 文件名 | 主要用途 |
|--------|----------|
| `summerOly_medal_counts.csv` | 核心预测目标与历史基准 |
| `summerOly_hosts.csv` | 提取"东道主效应"特征 |
| `summerOly_programs.csv` | 计算奖牌占比及归一化 |
| `summerOly_athletes.csv` | 分析项目优势与运动员规模 |
| `data_dictionary.csv` | 确保变量解析准确性 |

#### 2.3.2 数据清洗

1. **缺失值处理**：核心奖牌数据表无缺失记录。特定年份未获奖国家填充为0
2. **历史实体保留**：保留苏联、东德等历史记录，不合并至现代继承国
3. **文本字符清洗**：去除特殊空格字符，确保跨表关联准确

#### 2.3.3 特征工程

**滞后特征**：
$$\text{lag}_1(i,t) = \text{TotalMedals}_{i,t-4}$$

**滚动平均特征**：
$$\bar{M}_3(i,t) = \frac{1}{3}\sum_{k=1}^{3} \text{TotalMedals}_{i,t-4k}$$

**特征有效性验证**：

| 特征变量 | 描述 | 相关系数 | 重要性 |
|----------|------|----------|--------|
| `total_rolling3_mean` | 近3届平均奖牌数 | 0.835 | 非常高 |
| `total_lag1` | 上一届奖牌总数 | 0.800 | 非常高 |
| `gold_lag1` | 上一届金牌总数 | 0.791 | 非常高 |
| `is_host` | 是否为东道主 | 0.368 | 中等 |
| `participation_count` | 历史参赛届数 | 0.232 | 较低 |

## 第三章 问题一：奖牌预测模型

### 3.1 模型选择与方法论

#### 3.1.1 回归方法适用性论证

奥运奖牌预测本质上是数值预测问题，目标变量（金牌数、总奖牌数）为连续数值数据。采用回归分析而非分类方法的原因：

1. **目标变量连续性**：奖牌数为非负整数，属于连续数值预测范畴
2. **多因素影响**：奖牌分布受经济、人口、历史表现、东道主优势等多重因素共同作用
3. **数据可获取性**：所需特征变量在预测时点均已知，避免数据泄露

#### 3.1.2 探索性数据分析

目标变量分布分析发现：
- **右偏分布**：大多数国家奖牌数较少（中位数5枚），少数强国奖牌数异常高
- **均值-中位数差异**：平均12.5枚 vs 中位数5枚，数据被高奖牌国家拉高
- **历史趋势**：近三届平均奖牌数相对稳定

#### 3.1.3 东道主效应验证

**描述性统计对比**：
- 东道主平均获得66.9枚奖牌
- 非东道主平均11.3枚
- 增幅达491.9%

**独立样本t检验**：
$$t = \frac{\bar{X}_1 - \bar{X}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}} = 15.0$$

p值 < 0.001，拒绝原假设，东道主效应统计显著。

**效应量分析**（Cohen's d）：
$$d = \frac{\bar{X}_1 - \bar{X}_2}{s_{\text{pooled}}} = 2.77$$

根据Cohen标准（d > 0.8为大效应），东道主效应属于大效应范畴。

### 3.2 回归模型构建与评估

#### 3.2.1 线性回归与正则化方法

**多元线性回归**（基线模型）：
$$\hat{y} = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_7 x_7$$

**特征列表**：
- `total_lag1`：上届总奖牌数
- `total_lag2`：上两届总奖牌数
- `gold_lag1`：上届金牌数
- `total_rolling3_mean`：近三届平均
- `is_host`：东道主标识
- `total_events`：当届项目总数
- `participation_count`：参赛届数

**Lasso回归损失函数**（L1正则化）：
$$L = \sum_{i=1}^{m}(y_i - \hat{y}_i)^2 + \alpha \sum_{j=1}^{n}|\beta_j|$$

#### 3.2.2 集成学习模型

1. **随机森林**：通过构建多棵决策树取平均降低过拟合风险
   - 最重要特征：`gold_lag1`（0.345）、`total_lag1`（0.332）、`total_rolling3_mean`（0.151）

2. **梯度提升**：串行迭代方式，后一棵树纠正前一棵树的残差

#### 3.2.3 模型性能对比

| 模型 | RMSE | MAE | R² |
|------|------|-----|-----|
| 线性回归 | 4.58 | 3.00 | 0.945 |
| Ridge回归 | 4.58 | 3.00 | 0.945 |
| **Lasso回归** | **4.45** | **2.88** | **0.948** |
| 随机森林 | 5.20 | 3.17 | 0.930 |
| 梯度提升 | 4.60 | 2.97 | 0.945 |

**结论**：Lasso回归在测试集上表现最佳（R² = 0.948）

## 第四章 问题二："伟大教练"效应

### 4.1 方法论与证据检测

**核心挑战**：数据集中没有直接的教练信息！

因此采用**间接推断框架**，通过观察奖牌数据的模式变化推断教练效应，整合三种互补方法：

1. **突变点检测**：识别成绩的突然提升
2. **跨国奖牌流动分析**：追踪潜在的教练转会
3. **双重差分模型（DID）**：因果推断效应大小

**流程图**：见 `figures/flowchart_q2_method.svg`

#### 突变点检测标准

突变点判定：
$$\text{是突变点} = \begin{cases}\text{是} & \text{if } M_t - \bar{M}_{t-3:t-1} \geq 2 \text{ 且增幅} \geq 80\% \\\text{否} & \text{otherwise}\end{cases}$$

#### 跨国流动配对条件

$$\text{配对}(A, B) = \begin{cases}\text{是} & \text{if } \Delta M_A \leq -2 \text{ 且 } \Delta M_B \geq 2 \\\text{否} & \text{otherwise}\end{cases}$$

### 4.2 量化评估与案例验证

#### DID估计量

$$\hat{\tau}_{\text{DID}} = (\bar{M}_{T,\text{post}} - \bar{M}_{T,\text{pre}}) - (\bar{M}_{C,\text{post}} - \bar{M}_{C,\text{pre}})$$

**DID估计结果**：平均"伟大教练效应"可为受影响的特定国家-项目组合每个奥运周期贡献**2至5枚奖牌**。

#### 案例一：郎平与排球

- **2005-2008**：执教美国女排，美国获得2008北京奥运银牌
- **2013年后**：回归执教中国女排，带领中国获得2016里约奥运金牌

图示：`figures/fig2_langping_effect.svg`

#### 案例二：贝拉·卡罗伊与体操

- **1976-1981**：执教罗马尼亚，培养出科马内奇
- **1981年后**：叛逃美国，执教美国女子体操队
- **数据显示**：罗马尼亚呈下降趋势，美国急剧上升

图示：`figures/fig3_karolyi_effect.svg`

## 第五章 模型评估

### 5.1 优势

1. **数据驱动方法**：模型依赖128年、30届奥运会的广泛历史数据
2. **全面的特征工程**：多维特征矩阵捕捉短期表现和长期稳定性
3. **不确定性量化**：Bootstrap置信区间提供概率预测
4. **间接推断创新**：在缺乏直接数据时创造性地使用突变点检测和DID方法
5. **实际适用性**：框架为各国奥委会提供可操作的战略规划建议

### 5.2 局限性

1. **数据限制**：缺乏直接教练信息需要间接推断
2. **假设敏感性**：模型假设历史模式将继续，可能不适用于重大规则变化或地缘政治事件
3. **粒度限制**：当前预测在国家层面，项目级预测将提供更详细的洞察
