### 1.2 DID方法原理图

**用途**：解释差分-差分法的核心思想

**ASCII示意**：

```
        奖牌数
          ▲
          │
   After  │         ●─────────── 处理组（引入教练）
          │        /
          │       /    ← DID效应 = τ
          │      /
   Before │     ●────────────── 对照组（无变化）
          │    /
          │   /
          │  /
          │ ●
          │
          └──────────────────────────▶ 时间
                   │
              干预点（教练引入）
```

**绘图要点**：
1. 横轴为时间，纵轴为奖牌数
2. 两条线：处理组（实线）和对照组（虚线）
3. 干预点用**垂直虚线**标注
4. DID效应用**双向箭头**标注

## 二、参考文献检索指南

### 2.1 核心检索关键词

**英文关键词**：
1. "Great Coach Effect" + Olympic
2. Coach transfer + sports performance
3. Changepoint detection + time series + sports
4. Difference-in-differences + sports economics
5. "Lang Ping" + volleyball + coaching
6. "Béla Károlyi" + gymnastics
7. Sports investment + medal prediction
8. Home advantage + Olympic + coaching

**中文关键词**：
1. 教练效应 + 奥运会
2. 体育人才流动 + 国际比较
3. 突变点检测 + 体育数据
4. 差分-差分法 + 因果推断

### 2.2 推荐数据库

| 数据库 | 适用内容 | 访问方式 |
|--------|----------|----------|
| **Google Scholar** | 综合学术文献 | 免费 |
| **Web of Science** | 高影响力期刊 | 校园网 |
| **Scopus** | 体育科学文献 | 校园网 |
| **JSTOR** | 经济学方法论 | 校园网 |
| **中国知网** | 中文文献 | 校园网 |

### 2.3 重点参考文献方向

#### （1）教练效应相关

寻找关于教练对运动员/团队表现影响的量化研究：
- 教练转会对球队成绩的影响（足球、篮球领域较多）
- 教练培训投资与成绩提升的关系

#### （2）方法论相关

- **突变点检测**：时间序列分析中的changepoint detection方法
- **DID方法**：计量经济学中的差分-差分法应用案例
- **因果推断**：体育数据中的因果效应估计

#### （3）案例研究

- 郎平执教生涯的新闻报道和传记
- 贝拉·卡罗伊的纪录片和书籍
- 其他知名国际教练转会案例

### 2.4 引用格式要求

**参考MCM/ICM要求，使用以下格式**：

期刊文章：
```
[1] Author A, Author B. (Year). Title of article. Journal Name, Volume(Issue), Pages. DOI
```

书籍：
```
[2] Author A. (Year). Book Title. Publisher, City.
```

网站：
```
[3] Organization. (Year). Title of page. URL (Accessed: Date)
```

### 5.4 常见问题

**Q: 流程图用什么颜色？**
A: 建议使用项目标准配色：
- 主色：`#4682B4`（钢蓝）
- 辅色：`#FF7F50`（珊瑚红）
- 强调色：`#228B22`（森林绿）

**Q: DID图中的数值怎么填？**
A: 可以使用示意性数值，重点是展示**概念**而非精确数字。

**Q: 找不到相关文献怎么办？**
A: 可以用以下替代搜索：
- 把"Olympic"换成"sports"
- 把"coach effect"换成"management effect"或"leadership effect"
- 参考足球、篮球等职业体育的研究
