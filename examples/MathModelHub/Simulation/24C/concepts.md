| 题目中的专有名词 | 翻译 + 解释 |
|------------------|-------------|
| Momentum | 势头 - 指通过运动或一系列事件获得的力量/影响力，在网球比赛中体现为选手在特定阶段的优势延续或局势主导能力，是题目核心研究对象 |
| Grand Slam | 大满贯 - 网球界四项顶级赛事的统称，包括澳大利亚网球公开赛、法国网球公开赛、温布尔登网球锦标赛和美国网球公开赛，每项赛事为期两周，选手需在一年内赢得四项赛事冠军方可达成“大满贯”成就 |
| Wimbledon | 温布尔登网球锦标赛 - 四大满贯赛事之一，历史悠久，比赛场地为草地，题目数据均来自2023年该赛事男子单打第二轮后的比赛 |
| Gentlemen’s singles | 男子单打 - 温布尔登锦标赛的赛事项目之一，题目聚焦该项目的比赛数据，采用五局三胜制 |
| Set | 盘 - 网球比赛的基本单位之一，由若干局组成；赢6局可获胜一盘，但需至少领先2局；若战成6-6平，则需进行抢七决胜（第五盘抢七规则特殊） |
| Game | 局 - 由若干得分点组成，是盘的构成单位；赢4分可获胜一局，但需至少领先2分，存在“平分”“占先”等特殊得分状态 |
| Point | 得分点 - 网球比赛的最小竞争单位，每次发球至分出胜负为一个得分点，题目数据包含每一个得分点的详细信息（如发球方、获胜者、击球类型等） |
| Tie-breaker | 抢七 - 当一盘比赛双方战成6-6平时进行的决胜环节，温网规则为：除第五盘（决胜盘）采用10分制（需领先2分）外，其他盘采用7分制（需领先2分） |
| Deuce | 平分 - 局内得分达到40-40的状态，此时双方均至少获得3分，需其中一方连续赢得2分才能获胜该局 |
| Ad-in (Advantage in) | 发球方占先 - 平分后发球方赢得下一分的状态，再赢一分即可获胜该局 |
| Ad-out (Advantage out) | 接发球方占先 - 平分后接发球方赢得下一分的状态，再赢一分即可获胜该局 |
| Server | 发球方 - 每局比赛中发起首次击球的选手，职业网球中发球方具有明显优势，题目要求模型需考虑该因素 |
| Returner | 接发球方 - 每局比赛中接收对方发球的选手，需通过回球开启得分点竞争 |
| Breaking serve | 破发 - 接发球方赢得该局比赛，是比赛中局势反转的重要节点，可能影响“势头”变化 |
| Break point | 破发点 - 接发球方只要赢得该得分点即可获胜该局的关键节点，是比赛的重要转折点 |
| Holding serve | 保发 - 发球方赢得该局比赛，是维持自身优势、延续“势头”的重要方式 |
| Double fault | 双误 - 发球方两次发球均未将球发至有效发球区，判接发球方赢得该得分点 |
| Rally count | 回合数 - 一个得分点中双方选手合计击球的次数，反映该得分点的对抗强度 |
| Match_id | 比赛ID - 数据集中标识单场比赛的唯一编号（如“2023-wimbledon-1701”对应2023年温网男单决赛），用于筛选特定比赛数据 |
| Set_no / Game_no / Point_no | 盘号/局号/得分点编号 - 分别标识一个得分点所属的盘、局及在局内的顺序，用于定位比赛进程中的具体时刻 |
| Point victor | 得分点获胜者 - 赢得单个得分点的选手，是计算“势头”和比赛局势的核心数据项 |
| Game victor / Set victor | 局获胜者/盘获胜者 - 赢得某一局或某一盘的选手，用于判断比赛阶段性结果 |
| Serve_no | 发球次数 - 标识该得分点使用的是第一次发球（1）还是第二次发球（2），影响发球方的得分概率 |
| Winner shot type | 获胜击球类型 - 如“F”（Forehand，正手击球）、“B”（Backhand，反手击球）等，反映选手获胜的技术方式 |
| Break point won | 破发点成功 - 接发球方在破发点上获胜并赢得该局，是“势头”反转的重要指标 |
| Elapsed time | 耗时 - 从比赛第一个得分点开始到当前得分点发球的时间，用于定位比赛时间维度的进程 |
| Rally count | 回合数 - 单个得分点中双方选手的总击球次数，体现对抗的激烈程度和选手的相持能力 |
